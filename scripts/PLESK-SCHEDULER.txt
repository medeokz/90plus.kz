90plus.kz — ОДНА задача в Plesk
=================================

1) Git Pull на сервере (или загрузите новые scripts/)

2) Plesk → 90plus.kz → Scheduled Tasks → Add Task

   Type:     Fetch a URL
   URL:      https://90plus.kz/plesk-task.php?job=scheduler&key=ВАШ_DEPLOY_KEY
   Schedule: Every 2 minutes  (каждые 2 минуты)

3) Сохранить. Больше ничего вручную не нужно.

Что делает scheduler автоматически:
  premier-liga      каждые 5 мин
  world-cup         каждые 30 мин
  (матчи /games — вручную: ?job=fixtures-tracked&key=KEY)
  standings         каждые 30 мин
  articles          каждые 10 мин (1 источник за tick через Plesk, 6 локально)
  clubs-daily       раз в день после 04:00
  transfers         раз в день после 06:00

Время суток — часовой пояс сервера (проверьте APP_TIMEZONE в .env).

Проверка в браузере (один раз):
  https://90plus.kz/plesk-task.php?job=scheduler&key=KEY
  Ответ за 1–2 с: "scheduler: dispatched" — Plesk не ждёт парсер.
  Работа идёт в фоне (?job=worker). Лог: storage/logs/cron-scheduler.log

504 в Plesk на scheduler URL: сделайте git pull (spawn worker).
  Cron URL только ?job=scheduler — НЕ добавляйте ?job=worker в Plesk.
  Scheduler запускает только ОДНУ задачу за вызов worker.

Лог общий: storage/logs/cron-scheduler.log
Логи задач: storage/logs/cron-articles.log и т.д.

Deploy только после обновления кода:
  https://90plus.kz/plesk-task.php?deploy=1&key=KEY

Отдельные job=... URL по-прежнему работают (для ручного запуска).
